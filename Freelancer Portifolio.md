## **Selected Projects & Experience**

### **AI-Based Duplicate Payment & Fraud Detection Platform**

**Problem**  
Large ERP systems process high volumes of financial transactions, making them vulnerable to duplicate payments, fraud, and costly manual audits. Existing controls were largely reactive and rule-based, leading to false positives and delayed detection.

**Solution**  
Designed and built an AI-driven duplicate payment and fraud detection platform using Python and supervised machine learning models (Random Forest and Support Vector Classifier). The solution analyzed historical ERP payment data, vendor behavior, invoice patterns, and timing anomalies to proactively identify high-risk transactions.

**Architecture Highlights**

* ERP data ingestion via databases and APIs  
* Feature engineering and model training in Python  
* Scoring engine to flag suspicious transactions  
* Dashboards and reports for audit and finance teams

**Impact**

* Proactively identified performance degradation before outages occurred  
* Improved user experience by preventing slowdowns and system downtime  
* Achieved approximately 20% more accurate performance predictions compared to baseline methods

**AWS Architecture Bridge**  
This solution can be implemented on AWS using CloudWatch and custom metrics for data collection, Amazon S3 for historical data storage, AWS Lambda or ECS for forecasting jobs, and Amazon SageMaker for time-series modeling. Dashboards and alerts can be delivered via CloudWatch, Grafana, or Amazon QuickSight.
